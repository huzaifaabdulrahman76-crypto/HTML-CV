# 📄 HTML & CSS CV (Curriculum Vitae)

Welcome to my personal CV repository! This project is a fully responsive, modern web-based Curriculum Vitae designed using clean HTML and custom CSS.

## 🌐 Live Preview
You can view my live, fully-styled CV here:
👉 **[View My Live CV](https://huzaifaabdulrahman76-crypto.github.io/HTML-CV/)**

---

## 🚀 Features
* **Modern Layout:** Built with a clean, easy-to-read grid structure.
* **Fully Responsive:** Looks great on desktops, tablets, and mobile devices.
* **Semantic HTML:** Structured using modern HTML5 elements.
* **Separated Styling:** Styles are cleanly organized in an external `style.css` stylesheet (keeping the HTML file lightweight and readable!).

## 🛠️ Technologies Used
* **HTML5** - For structure and markup.
* **CSS3** - For custom layouts, typography, and color schemes.
* **GitHub Pages** - For live hosting.
