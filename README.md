A modern, responsive **Curriculum Vitae (CV)** website built using **HTML5** and **CSS3**. This project showcases my education, technical skills, projects, and personal information through a clean, professional, and mobile-friendly web design.

---

## 📖 About

This project was created to design a professional online CV using only HTML and CSS. It focuses on responsive web design, semantic HTML, and modern CSS styling while serving as a digital portfolio that can be viewed from any device.

---

## ✨ Features

- 📱 Fully responsive design
- 🎨 Modern and clean UI
- 📄 Professional CV layout
- 🧱 Semantic HTML5 structure
- 🎯 External CSS styling
- 💼 Education, skills, and projects section
- ⚡ Lightweight and fast loading
